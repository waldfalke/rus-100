# (Устаревшее) Предложение по типографической системе Rus100

# Типографическая система для образовательного сайта по русскому языку

## 1. Шрифтовая пара

Сохраняя Inter как основной шрифт для интерфейса, предлагаю следующую шрифтовую пару:

### Основной шрифт (для интерфейса и основного текста)

- **Inter** (400, 500, 600, 700)
- Фоллбэки: Arial, Helvetica, sans-serif

### Дополнительный шрифт (для заголовков и акцентов)

- **Bitter**
- Фоллбэки: Georgia, serif

**Обоснование:**

1. **Bitter** — это современный серифный шрифт с отличной поддержкой кириллицы, разработанный специально для экранного использования
2. Создает ясный контраст с sans-serif шрифтом Inter, что укрепляет иерархию
3. Имеет превосходную читаемость даже при небольших размерах
4. Академический характер серифного шрифта хорошо соответствует образовательному контексту
5. Bitter имеет хорошо разработанные начертания для полужирного и курсивного текста, что позволяет гибко выделять важную информацию

## 2. Модифицированная шкала размеров

Предлагаю скорректированную шкалу размеров для улучшения читаемости, учитывая особенности целевой аудитории:

```
xs: 0.875rem (14px) — ↑ с 12px до 14px
sm: 1rem (16px) — ↑ с 14px до 16px
base: 1.125rem (18px) — ↑ с 16px до 18px
lg: 1.25rem (20px) — ↑ с 18px до 20px
xl: 1.5rem (24px) — ↑ с 20px до 24px
2xl: 1.75rem (28px) — ↑ с 24px до 28px
3xl: 2rem (32px) — ↓ с 30px до 32px
4xl: 2.5rem (40px) — без изменений
5xl: 3rem (48px) — без изменений
6xl: 3.75rem (60px) — без изменений
```

**Обоснование изменений:**

1. **Увеличение минимального размера** (xs) с 12px до 14px улучшает читаемость для пользователей с ухудшенным зрением (в т.ч. учителей старшего возраста)
2. **Увеличение базового размера** с 16px до 18px соответствует современным тенденциям и улучшает читаемость текстовых блоков с правилами русского языка
3. **Более плавное масштабирование** между меньшими размерами обеспечивает лучшую иерархию информации
4. **Сохранение крупных размеров** для главных заголовков поддерживает визуальную иерархию

## 3. Рекомендации по применению

### Для основного текста

- **Шрифт:** Inter
- **Размер:** base (18px)
- **Вес:** Regular (400) или Medium (500)
- **Пример применения в Tailwind:** `font-inter text-base font-normal text-gray-900`
- **Для правил языка:** Увеличить контраст, используя более темный оттенок текста

### Для интерактивных элементов (пилюли/бейджи)

- **Шрифт:** Inter
- **Размер:** sm (16px) или xs (14px) для компактных элементов
- **Вес:** Medium (500) или Semibold (600)
- **Пример применения в Tailwind:** `font-inter text-sm font-medium bg-indigo-100 text-indigo-800 rounded-full px-3 py-1`
- **Для числовых значений внутри бейджей:** Использовать табличные цифры для лучшего выравнивания `font-feature-settings: "tnum"`

### Для заголовков

- **Шрифт:** Bitter
- **Размеры и веса:**
    - H1: 4xl (40px), Bold (700)
    - H2: 3xl (32px), Bold (700)
    - H3: 2xl (28px), Semibold (600)
    - H4: xl (24px), Semibold (600)
    - H5: lg (20px), Medium (500)
    - H6: base (18px), Medium (500)
- **Пример применения в Tailwind:** `font-bitter text-3xl font-bold text-gray-900 mb-4`

### Для вспомогательного текста

- **Шрифт:** Inter
- **Размер:** sm (16px) или xs (14px)
- **Вес:** Regular (400)
- **Пример применения в Tailwind:** `font-inter text-sm font-normal text-gray-600`
- **Для метаданных и подсказок:** Использовать более светлый оттенок, но с достаточным контрастом для доступности

## 4. Высота строк и отступы

### Высота строк

- **Для основного текста:** 1.6 (оптимально для кириллицы)
- **Для заголовков:** 1.2-1.3 (более плотная)
- **Для компактных элементов интерфейса:** 1.4
- **Для расширенных текстовых блоков с правилами:** 1.7

**Пример в Tailwind:**

```
Основной текст: leading-relaxed (1.625)
Заголовки: leading-tight (1.25)
Компактные элементы: leading-normal (1.5)
Расширенные блоки: leading-loose (2) или custom (1.7)
```

### Отступы между абзацами

- **Между абзацами:** 1.5em (относительно размера текста)
- **Между заголовком и текстом:** 0.75em
- **Между разделами:** 2em

**Пример в Tailwind:**

- Между абзацами: `mb-6` (для базового размера 18px)
- После заголовка: `mb-3`
- Между разделами: `mb-12`

## 5. Адаптивность

### Мобильные устройства (< 640px)

- **Основной текст:** Уменьшить до 16px (sm в новой шкале)
- **Заголовки:** Уменьшить на один шаг в иерархии
    - H1: 3xl (32px)
    - H2: 2xl (28px)
    - H3: xl (24px)
    - H4: lg (20px)
- **Интерлиньяж:** Увеличить на 0.1-0.2 для лучшей читаемости

### Планшеты (640px - 1024px)

- **Основной текст:** Базовый размер (18px)
- **Заголовки:** Стандартная шкала
- **Интерлиньяж:** Стандартный

### Десктопы (> 1024px)

- **Основной текст:** Базовый размер (18px) или увеличенный (20px) для более широких экранов
- **Заголовки:** Стандартная шкала или увеличенная для крупных мониторов
- **Максимальная ширина текстового контейнера:** 70ch для оптимальной читаемости

## 7. Преимущества предложенной типографической системы

1. **Улучшенная читаемость**
    
    - Увеличенный базовый размер шрифта (18px) соответствует современным стандартам и облегчает чтение длинных текстов с правилами русского языка
    - Оптимизированный интерлиньяж 1.6 для кириллического текста обеспечивает лучшее восприятие букв с высокими выносными элементами
2. **Ясная иерархия информации**
    
    - Контрастная шрифтовая пара (sans-serif/serif) создает визуальное разделение между интерфейсом и контентом
    - Более последовательная шкала размеров с равномерными шагами позволяет точнее контролировать важность элементов
3. **Культурное соответствие**
    
    - Bitter имеет хорошо проработанные кириллические символы, сохраняющие академический характер
    - Inter обеспечивает нейтральность и современный вид интерфейса, сохраняя при этом высокую разборчивость кириллицы
4. **Техническая оптимизация**
    
    - Минимальное количество шрифтов (2) и начертаний (4 для каждого) способствуют быстрой загрузке
    - Полная совместимость с Tailwind CSS через настройку конфигурации
    - Рекомендуемые фоллбэки обеспечивают стабильное отображение даже при проблемах с загрузкой шрифтов
5. **Инклюзивность и доступность**
    
    - Увеличенные минимальные размеры (14px вместо 12px) помогают пользователям с ухудшенным зрением
    - Достаточные контрасты для всех типов текста соответствуют стандартам WCAG

## 8. Итоговые рекомендации по внедрению

1. **Поэтапное внедрение**
    
    - Сначала обновите шрифты и их конфигурацию в проекте
    - Затем модифицируйте базовые компоненты (заголовки, параграфы)
    - Наконец, обновите специфичные UI-элементы (бейджи, карточки с правилами)
2. **Создание компонентов**
    
    - Разработайте набор типографических компонентов для согласованного использования в проекте
    - Документируйте использование компонентов для разработчиков
3. **A/B-тестирование**
    
    - Протестируйте новую типографику на ограниченной аудитории
    - Соберите обратную связь от учителей старшего возраста для оценки читаемости
4. **Производительность**
    
    - Используйте `font-display: swap` для оптимизации видимого текста до загрузки шрифтов
    - Предзагружайте только необходимые начертания для первого экрана (например, preconnect для Google Fonts)
5. **Техническая интеграция**
    
    - Расширение существующих компонентов с помощью новых классов без нарушения обратной совместимости
    - Документация использования (styleguide) для разработчиков и дизайнеров

## 11. Заключение

Предложенная типографическая система для образовательного сайта по русскому языку основана на глубоком понимании целевой аудитории и специфики контента. Ключевые принципы системы:

1. **Баланс между техническими ограничениями и потребностями пользователей**
    
    - Минимальное количество шрифтов для быстрой загрузки
    - Увеличенные базовые размеры для лучшей читаемости
    - Оптимизированная для кириллицы высота строк
2. **Шрифтовая система, поддерживающая иерархию контента**
    
    - Inter как современный и нейтральный шрифт для интерфейса
    - Bitter как читаемый серифный шрифт для заголовков, создающий академическую атмосферу
    - Тщательно подобранные размеры и веса для четкой визуальной иерархии
3. **Адаптивный подход**
    
    - Простые, но эффективные изменения размеров для разных устройств
    - Сохранение читаемости на всех экранах
    - Специальная оптимизация для мобильных устройств
4. **Интеграция с технической базой**
    
    - Полная совместимость с Tailwind CSS
    - Компонентный подход для устойчивой типографической системы
    - Оптимизированная производительность шрифтов

Эта система создает прочную основу для образовательного сайта, делая учебный контент максимально доступным и приятным для чтения, сочетая технические преимущества с высоким уровнем эстетики и удобства использования.

Превосходная типографическая система создает не просто читаемый текст, но и формирует доверие к образовательной платформе, что критически важно для пользователей, изучающих русский язык.
